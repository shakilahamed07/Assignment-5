Question-1: What are the different ways to select an element in the DOM?

Document.getElementById() and Document.querySelector() can be used to select an element in the DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used when you want to access or modify the HTML content inside an element. InnerText is used only when you want to access or modify the visible text content of the element. To access or modify all the text content of an element, whether it is visible or hidden, textContent is used.

Question-3: What is event delegation in the DOM?

Event Delegation is a powerful event handling technique in JavaScript where a single event listener is used for multiple elements of the same type. Through this technique we can handle the events of its child elements by adding an event listener to the parent element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism for event propagation in the DOM (Document Object Model). When an event occurs on an element (such as a click event), that event bubbles up to the top of the DOM tree, i.e. reaching the parent elements. This process is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

A new HTML element can be created using the document.createElement() method. appendChild() method can be used to add new child to parent. An element can be removed using removeChild() methods.
